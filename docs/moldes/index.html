Tutorial de moldes

En este tutorial se muestra paso a paso como crear un molde para un objeto usando de herramienta el programa Fusion 360.

Se comienza realizando un sketch de la forma que se quiere realizar, dandole un eje de simetría, la cual es la línea punteada

Luego se genera un revolve del dibujo con eje de simetría la línea punteada, generando la figura que se muestra a continuación.

Tras esto, se dibuja un cubo que contenga la figura ya hecha.

Extruimos el cubo y lo cortamos con la figura ya creada, dejando así el negativo de esta.

Por último, generamos una pequeña inclinación en la superficie para poder sacar el molde de forma más sencilla cuando se realice.

Se adjunta video con los pasos.
