Investigación de Materiales

En esta sección se realizará una investigación sobre materiales a modo de experimentación.

El encargo:

El encargo consistía en realizar una investigación de materiales en base a papeles y cartones reciclados, a modo de realizar diferentes maceteros con diferentes mix de materiales.

¿Qué es el papel maché?:

El papel maché consiste en un material bastante simple, compuesto de papel (o cartón) triturado y humedecido para que sea simple de moldear. Generalmente, a esta mezcla se le agrega cola fría para lograr una mejor adehesión entre el mismo papel, para así generar una forma más dura. Con esta técnica se pueden realizar miles de actividades para todo tipo de personas, por lo que tiene un gran potencial al ser muy simple de hacer y usar. Aún así, en este caso, no se utilizará el material de cola fría, por lo que se reemplazará con otros materiales, a modo de experimentar cuál es el más eficiente.

Los materiales:

Como primera tentativa de materiales, decidí usar hojas usadas de cuaderno, para darle un segundo uso a estas, y para reemplazar la cola fría, usé clara de huevo debido a que es un muy buen adehesivo natural.

El proceso:

Como primer paso, lo primero que realicé fue dejar el papel remojando un día completo en agua para que así quedara más fácil de manipular.

Tras esto, empece a romper el papel en pedazos pequeños (de forma manual) para que así fuera quedando una especie de pasta más moldeable.

Luego, pasé esta pasta por un colador para que no quedara un exceso de agua y su proceso de secado fuera más rapido.

Después de esto, le puse un huevo, le saqué la yema y comencé a revolver la mezcla para que quedara repartido de forma uniforme.

Ahora lo único que quedaba era encontrar un recipiente para darle la forma y esperar a que se secara al sol. (por problemas técnicos un gato me botó el primer intento y lo tuve que hacer denuevo)

Y seguimos a la espera de que se seque.
