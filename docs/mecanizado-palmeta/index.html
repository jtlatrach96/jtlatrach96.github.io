Bonus: Mecanizado Palmeta

En esta sección se mecanizará una palmeta para que esta sirva posteriormente como un molde.

¿Qué es el CNC?:

El CNC o corte numérico por computadora es un sistema que permite controlar la posicion de un elemento físico, generalmente una herramienta que está montada en una máquina. Esta se maneja utilizando los ejes X Y Z para moverse. A continuación se adjunta un ejemplo de una máquina CNC.

¿Por qué 2.5D?:

El mecanizado 2.5D hace referencia a una superficie que esuna proyección de un plano en tercera dimension, por lo que, a pesar de que el objeto sea de 3 dimensiones, no hay elementos que sobresalen. Los objetos de este tipo son representados generalmente como un mapa de contorno que da altura (grosor o profundidad) del objeto en cada punto. Fuente: es.qwe.wiki / Mecanizado 2.5D

Modelo a construir:

El modelo a realizar es la estructura para la fabricación de un molde.

Proceso de fabricación:

Comenzamos generando una base de un rectángulo extruido que servirá como el pedazo de madera en este caso del que vamos a comenzar a generar nuestro respaldo. Cabe recalcar que es importante que la extrusión contenga a TODO el modelo, este se ve en la imagen de forma transparente.

Para comenzar, nos ubicamos en la vista top del modelo, generamos un sketch en esta vista y proyectamos su área para luego trabajar sólo en esa sección.

Luego, nos vamos a la sección de fabricación y elegimos la opción de Adaptative Clearing, que es básicamente un sacado de forma más bien rápida y en bruto, básicamente generando un desbaste del material para luego trabajarlo a detalle (en las opciones seleccionamos el boundary que creamos anteriormente)

Tras un tiempo el proceso de desbaste termina viendose así:

Ahora, para trabajar más en detalle, elegimos la opción Parallel, donde hace un pasado más fino y detallado al material. Seleccionamos básicamente lo mismo que en la opción de desbaste y apretamos OK.

Tras haber realizado el proceso se verá de la siguiente manera, pero este es sólo un pase en una dirección.

Para realizar el pase en la otra dirección, duplicamos el proceso en la parte izquierda y lo editamos.

En en el tab de Passes le damos un degree de 90°, lo que hará que este pase en la otra dirección que no habíamos realizado.

Así se verá tras generarlo en ambas direcciones.
